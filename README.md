Kissan Mitra 🌾 - Maharashtra Farming Guide
Kissan Mitra (Farmer's Friend) is a web-based decision support system designed specifically for farmers in Maharashtra, India. It helps farmers make data-driven decisions about crop selection, fertilizer usage, and seasonal planning to maximize yield and income.

🚀 Features
Smart Crop Recommendation: Input your farm's specific details (soil type, water availability, acreage, temperature, and pH) to receive the top 3 most suitable crops.

Step-by-Step Farming Guides: Detailed instructions for each recommended crop, from nursery preparation to harvest.

Fertilizer Management: A comprehensive database of Nitrogen (Urea), Phosphorus (DAP), and Potassium (MOP) requirements for various crops.

Maharashtra Centric: Tailored for the specific soil types (Black, Red, Laterite) and seasons (Kharif, Rabi, Summer) found in the Maharashtra region.

User-Friendly Interface: Simple, accessible design with clear icons and mobile-responsive layouts.

🛠️ Technology Stack
HTML5: Semantic structure for accessibility.

CSS3: Custom styling with a focus on responsive design and modern UI components (Cards, Grids, Flowcharts).

JavaScript (ES6+): Logic for crop prediction algorithms and dynamic UI section switching.

📊 How the Prediction Works
The application uses a scoring algorithm based on several environmental factors:

Soil Compatibility: Matches crop requirements against user soil color/type.

Water & Rainfall: Evaluates if the irrigation level and annual rainfall meet the crop's threshold.

Climate Control: Checks if the local temperature range falls within the crop's optimal growing window.

Soil Health: Validates the pH level for the specific crop species.

Crops are ranked by a Suitability Score (0-100), and the top 3 matches are displayed with specific market price estimates and yield expectations.
